# <a name="keyboard"></a>키보드

![키보드](images/UX_Hero_Keyboard.jpg)

혼합 현실 응용 프로그램은 언제 든 지 시스템 키보드를 호출할 수 있으며,이는 대상 플랫폼의 기능에 따라 동작 합니다. 예를 들어 HoloLens 2 키보드는 직접 상호 작용을 지원 하지만 HoloLens (첫 번째 gen) 키보드는 응시 및 공기 탭을 지원 합니다.
