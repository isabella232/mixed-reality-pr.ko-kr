# <a name="tooltip"></a>도구 설명

![도구 설명](images/UX_Hero_Tooltip.jpg)

도구 설명은 다른 컨트롤이나 개체에 연결된 간단한 설명입니다. 자세히 검사할 때 힌트 또는 추가 정보를 전달하는 데 사용됩니다. 도구 설명은 사용자가 직접 설명되지 않은 물리적 환경에서 익숙하지 않은 개체를 이해하는 데 도움이 될 수 있습니다.
