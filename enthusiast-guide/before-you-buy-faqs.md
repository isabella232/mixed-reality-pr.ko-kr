**빠른 답변:** Windows Mixed Reality PC 및 Ultra PC에 대한 자세한 내용은 다음에서 캡처됩니다.<https://aka.ms/mrcompat>

### <a name="why-is-this-called-windows-mixed-reality-when-the-devices-look-like-they-provide-a-virtual-reality-experience"></a>디바이스가 "가상 현실" 환경을 제공하는 것처럼 보일 때 이를 "Mixed Reality"Windows 이라고 하는 이유

**빠른 답변:** Mixed Reality 증강 현실 및 홀로그램에서 가상 현실에 이르는 공간 컴퓨팅의 전체 스펙트럼을 나타냅니다. Windows Mixed Reality 플랫폼은 VR(Virtual Reality) 및 AR(증강 현실)에서 디바이스를 지원합니다. 현재 해당 스펙트럼에서 유사한 기술을 가진 두 가지 디바이스 유형인 AR(HoloLens) 및 모션 컨트롤러가 있는 VR(Windows Mixed Reality) 헤드셋을 지원합니다.

Mixed Reality 대해 자세히 알아보세요. </windows/mixed-reality/mixed-reality>

### <a name="what-is-the-difference-between-windows-mixed-reality-and-other-vr-headsets"></a>Windows Mixed Reality 다른 VR 헤드셋의 차이점은 무엇인가요?

**빠른 답변:** Windows Mixed Reality 헤드셋은 내부 추적 기능(추적 카메라가 헤드셋에 있습니다)과 Windows 10 및 Windows 11을 사용하여 플러그 앤 플레이 설정을 제공합니다.

**자세한 정보:** Windows Mixed Reality 간편한 설정(외부 센서가 사용자와 컨트롤러를 추적할 필요 없음), 편리하고 가격 대비 좋은 헤드셋 선택, 경쟁력 있는 해상도를 갖춘 헤드셋, 마지막으로 공간 인터페이스를 제공하는 고유한 사용자 환경을 포함하여 Microsoft Store 수천 개의 앱을 사용할 수 있는 몇 가지 기능을 제공합니다.

### <a name="does-mixed-reality-mean-that-inside-out-cameras-are-passthrough-can-you-experience-augmented-reality-in-addition-to-virtual-reality"></a>Mixed Reality 내부 카메라가 통과되는 것을 의미하나요? 가상 현실 외에 증강 현실도 경험할 수 있나요?

**빠른 답변:** 아니요, Windows Mixed Reality 헤드셋의 내부 카메라는 위치 추적에만 사용됩니다. Windows Mixed Reality 헤드셋은 가상 현실용이며 실제 세계 또는 증강 현실에 대한 보기를 제공하지 않는다는 것을 의미합니다.

### <a name="what-is-inside-out-tracking-how-is-it-different-than-outside-in-tracking-or-lighthouse-tracking"></a>내부 추적이란? 외부 추적 또는 등대 추적과 어떻게 다른가요?

* **내부 추적** Windows Mixed Reality, Oculus Quest 및 Vive/Index 시스템은 내부 추적을 사용합니다. 내부 아웃 추적 카메라는 헤드셋에 기본 제공되고 환경의 변경 내용을 추적하여 이동할 때 환경에 따라 헤드셋의 위치를 결정합니다. HTC Vive 및 밸브 인덱스와 같은 일부 시스템은 카메라 대신 센서를 사용하고 외부 "Lighthouse" 또는 "기본 스테이션"에 의존하여 환경을 추적합니다.

* **외부 추적** OculusStation 및 PlayStation VR과 같은 시스템은 외부 추적을 사용합니다.  외부 추적을 통해 헤드셋은 하나 이상의 외부 디바이스에서 추적됩니다. 카메라는 이러한 외부 디바이스에 기본 제공되고 환경 주위에 배치되며 환경에 따라 헤드셋의 위치를 결정하는 데 사용됩니다.

[내부](./tracking-system.md)추적 에 대해 자세히 알아보세요.

### <a name="can-inside-out-tracking-of-motion-controllers-impact-my-game-play-due-to-camera-fov"></a>카메라 FOV로 인해 모션 컨트롤러의 내부 추적이 내 게임 플레이에 영향을 줄 수 있습니다.

 헤드셋과 마찬가지로 Windows Mixed Reality 모션 컨트롤러에는 외부 추적 센서를 설치할 필요가 없습니다. 대신 컨트롤러는 헤드셋 자체의 센서에 의해 추적됩니다. 사용자가 헤드셋의 보기 필드에서 컨트롤러를 이동하는 경우 대부분의 경우 Windows 컨트롤러 위치를 계속 유추하여 앱에 제공합니다. 컨트롤러가 오랫동안 시각적 추적을 손실한 경우 컨트롤러의 위치는 근사 정확도 위치로 떨어집니다. 이 시점에서 시스템은 내부 방향 센서를 사용하여 컨트롤러의 진정한 방향을 계속 노출하면서 사용자가 이동할 때 사용자의 위치를 추적하여 컨트롤러를 사용자에게 본문 잠금합니다. 컨트롤러를 사용하여 UI 요소를 가리키고 활성화하는 많은 앱은 사용자가 모르게 대략적인 정확도로 정상적으로 작동할 수 있습니다. 이에 대한 느낌을 얻는 가장 좋은 방법은 직접 사용해보는 것입니다.

### <a name="how-is-windows-mixed-reality-different-from-samsung-gear-vr"></a>Samsung Gear VR과 Windows Mixed Reality 어떻게 다른가요?

**빠른 답변:** Samsung Gear VR 및 Google Daydream을 비롯한 모든 스마트폰 기반 VR 제품은 3DOF(3DOF) 헤드 위치 추적 시스템을 사용합니다. 3DOF 기반 시스템을 사용하면 가상 세계에서 위로 **머리만** 이동할 수 있습니다. Windows 반면 MR는 6DOF(6도 자유도) 헤드 및 컨트롤러 위치 추적 시스템을 사용합니다. 6DOF 기반 시스템을 사용하면 가상 세계에서 본문 전체를 이동할 수 있습니다. 따라서 실제 세계에서처럼 개체를 둘러보고 다양한 관점에서 볼 수 있습니다. 또한 6DOF 기반 시스템이기도 합니다.

* 3DOF 및 6DOF에 대한 자세한 내용은 다음을 참조하세요. </windows/mixed-reality/fake-guide/tracking-system#what-is-the-difference-between-3dof-and-6dof> 
* 여기에서 위치 추적 및 DOF(Degree's of Freedom)에 대해 자세히 읽어보세요. <https://www.roadtovr.com/introduction-positional-tracking-degrees-freedom-dof/>

### <a name="does-mixed-reality-mean-that-the-inside-out-cameras-are-passthrough"></a>"Mixed Reality"는 내부 외부 카메라가 통과되는 것을 의미하나요?

### <a name="do-the-mixed-reality-headsets-have-ipd-adjustment"></a>혼합 현실 헤드셋에 IPD 조정이 있나요?

**빠른 답변:** 헤드셋에 따라 일부 혼합 현실 헤드셋은 기계식 IPD 조정 기능을 제공합니다. 다른 헤드셋은 IPD에 따라 이미지 왜곡 및 깊이 정확도를 향상시키는 소프트웨어 IPD 조정 기능을 제공합니다. 사용자는 설정 > Mixed Reality > 헤드셋 디스플레이 **> 보정을** 통해 사용자 지정 IPD를 설정할 수 있습니다.

### <a name="do-the-mixed-reality-headsets-have-eye-relief-adjustment"></a>혼합 현실 헤드셋에 시선 조절 조정이 있나요?

**빠른 답변:** 아니요, Windows Mixed Reality 헤드셋에는 시선 조절 조정이 없습니다.

### <a name="will-there-be-issues-with-motion-controller-tracking-if-multiple-windows-mixed-reality-headsets-are-in-close-proximity"></a>여러 Windows Mixed Reality 헤드셋이 근접한 경우 모션 컨트롤러 추적에 문제가 있나요?

**빠른 답변:** 모션 컨트롤러에 간섭이 없는 것을 발견했지만 혼합 현실에서 최상의 환경을 위한 충분한 공간을 사용자에게 제공해야 합니다(예: 10피트 x 10피트).

### <a name="is-there-boundary-chaperone-or-guardian-system-in-windows-mixed-reality"></a>Windows Mixed Reality 경계, chaperone 또는 보호자 시스템이 있나요?

**빠른 답변:** Windows Mixed Reality 경계를 설정할 수 있습니다. 또한 Windows MR 헤드셋 중 일부에는 전면 힌지 디스플레이가 있으므로 작업하는 동안 헤드셋을 위로 전환할 수 있습니다.

### <a name="do-windows-mixed-reality-headsets-work-in-the-dark"></a>Windows Mixed Reality 헤드셋이 어둡게 작동합니까?

**빠른 답변:** 너무 어둡기 때문에 실내를 자신 있게 이동할 수 없는 경우 추적 시스템이 해당 환경에서 작업하는 데 어려움을 겪는 것이 좋은 일반적인 규칙입니다.

### <a name="what-is-the-cable-length-of-the-windows-mixed-reality-headset"></a>Windows Mixed Reality 헤드셋의 케이블 길이는 무엇인가요?

**빠른 답변:** 일반적으로 Windows Mixed Reality 헤드셋의 케이블 길이는 4미터이지만 헤드셋에 따라 달라집니다. 자세한 내용은 다음을 확인하세요. <https://www.microsoft.com/en-us/store/collections/vrandmixedrealityheadsets>

### <a name="can-i-use-a-usb--hdmi-extension-cable-with-windows-mixed-reality-headsets"></a>Windows Mixed Reality 헤드셋과 함께 USB/HDMI 확장 케이블을 사용할 수 있나요?

**빠른 답변:** 확장 코드 없이 작동하도록 Windows Mixed Reality 설계했습니다. 혼합 현실 헤드셋에서 확장 코드를 사용하는 것은 지원되지 않으며 사용은 사용자 경험에 영향을 미칠 수 있습니다.

### <a name="will-my-pc-work-with-windows-mixed-reality-what-are-the-minimum-specs"></a>내 PC가 Windows Mixed Reality 작동합니까? 최소 사양은 무엇인가요?

**빠른 답변:** [여기에서](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md) 최소 사양을 찾거나 Mixed Reality 포털 다운로드하여 실행하여 PC가 [Windows Mixed Reality](https://www.microsoft.com/p/mixed-reality-portal/9ng1h8b3zc7m?activetab=pivot:overviewtab) 작동하는지 확인할 수 있습니다.

### <a name="will-windows-mixed-reality-work-with-my-xbox"></a>내 Xbox에서 Windows Mixed Reality 작동합니까?

## <a name="motion-controllers"></a>모션 컨트롤러

### <a name="how-do-motion-controllers-work-with-my-pc"></a>내 PC에서 모션 컨트롤러는 어떻게 작동하나요?

**빠른 답변:** Windows Mixed Reality 모션 컨트롤러는 Bluetooth 사용합니다. 헤드셋에 따라 모션 컨트롤러는 헤드셋의 Bluetooth 라디오(장착된 경우) 또는 PC의 Bluetooth 라디오에 페어링됩니다.

* Acer OJO 500, Samsung Odyssey+, HP Reverb 및 HP Reverb G2 기능을 포함한 일부 Windows Mixed Reality 헤드셋은 모션 컨트롤러에서 사용할 수 있는 기본 제공 Bluetooth 라디오입니다. 이러한 헤드셋과 함께 제공되는 모션 컨트롤러는 팩터리에서 헤드셋에 미리 페어링되며 PC에 별도의 Bluetooth 라디오가 필요하지 않습니다.
* 다른 Windows Mixed Reality 헤드셋은 PC의 Bluetooth 라디오와 페어링해야 합니다.

### <a name="are-windows-mixed-reality-motion-controllers-cross-compatible-between-windows-mixed-reality-headsets"></a>Windows Mixed Reality 모션 컨트롤러는 Windows Mixed Reality 헤드셋 간에 상호 호환합니까?

**빠른 답변:** 예, Windows Mixed Reality 모션 컨트롤러는 모든 Windows Mixed Reality 헤드셋에서 작동합니다.

### <a name="how-do-i-connect-motion-controllers-if-i-do-not-have-built-in-bluetooth"></a>기본 제공 Bluetooth 없는 경우 모션 컨트롤러를 연결할 어떻게 할까요? 있나요?

**빠른 답변:** PC에 기본 제공 Bluetooth 지원이 없는 경우 Bluetooth 4.0을 지원하는 USB Bluetooth 어댑터를 연결하여 모션 컨트롤러를 사용하도록 설정해야 합니다.

### <a name="will-the-controllers-work-with-bluetooth-31-or-do-i-need-bluetooth-40"></a>컨트롤러가 Bluetooth 3.1에서 작동합니까 아니면 Bluetooth 4.0이 필요한가요?

**빠른 답변:** Windows Mixed Reality 지원되는 최소 사양은 Bluetooth 4.0입니다. PC에 Bluetooth 3.1이 있는 경우 컨트롤러 사용이 차단되지 않지만 가장 좋은 환경은 BT 4.0에 있습니다. [Mixed Reality 포털](https://www.microsoft.com/p/mixed-reality-portal/9ng1h8b3zc7m?activetab=pivot:overviewtab) 다운로드하고 실행하여 PC가 WMR 준비가 되었는지 확인합니다.

### <a name="will-windows-mixed-reality-controllers-work-with-hololens"></a>Windows Mixed Reality 컨트롤러가 HoloLens 작동합니까?

**빠른 답변:** 아니요, Windows Mixed Reality 모션 컨트롤러는 HoloLens 작동하지 않습니다.

### <a name="will-windows-mixed-reality-work-with-my-oculus-touch-controllers-or-htc-vive-headsets--controllers"></a>Windows Mixed Reality Oculus Touch 컨트롤러 또는 HTC Vive 헤드셋/컨트롤러와 함께 작동하나요?

**빠른 답변:** 아니요, Windows Mixed Reality 승인된 헤드셋/컨트롤러에서만 작동합니다. <https://www.microsoft.com/store/collections/VRandMixedrealityheadsets>을 참조하십시오.

### <a name="will-windows-mixed-reality-work-with-my-steamvr-knuckle-controllers"></a>내 SteamVR Knuckle 컨트롤러에서 Windows Mixed Reality 작동합니까?

**빠른 답변:** 아니요, Windows Mixed Reality 승인된 헤드셋/컨트롤러에서만 작동합니다. <https://www.microsoft.com/store/collections/VRandMixedrealityheadsets>을 참조하십시오.

### <a name="will-windows-mixed-reality-support-my-xbox-one-controller-with-the-wireless-xbox-adapter"></a>Windows Mixed Reality 무선 Xbox 어댑터를 Xbox One 컨트롤러를 지원합니까?

**빠른 답변:** 예, 무선 Xbox 어댑터에서 작동합니다(Bluetooth 필요 없음).

## <a name="comfort"></a>편안함

### <a name="can-i-wear-a-windows-mixed-reality-headset-with-glasses"></a>Windows Mixed Reality 헤드셋을 사용할 수 있나요?

**빠른 답변:** 예, Windows Mixed Reality 헤드셋을 안경과 없이 모두 사용할 수 있습니다.

### <a name="does-windows-mixed-reality-support-productivity-within-a-vr-headset-on-a-desktop-environment"></a>Windows Mixed Reality 데스크톱 환경의 VR 헤드셋 내에서 생산성을 지원하나요?

**빠른 답변:** Windows Mixed Reality 헤드셋 내에서 Mail & Calendar와 같은 Windows UWP 앱을 멀티태스킹하고 데스크톱 미리 보기 앱을 사용하여 데스크톱에서 실행되는 전체 Win32 앱에 액세스할 수 있습니다.

### <a name="can-i-see-my-keyboard-from-inside-a-windows-mixed-reality-headset"></a>Windows Mixed Reality 헤드셋 내에서 키보드를 볼 수 있나요?

**빠른 답변:** 헤드셋에서 실제 키보드를 볼 수는 없지만 소프트웨어 키보드를 사용할 수 있습니다. 또한 많은 Windows Mixed Reality 헤드셋에는 전면 힌지 디스플레이가 있으므로 작업하는 동안 헤드셋을 위로 대칭 이동하면 됩니다. 자세한 내용은 다음을 수행합니다. <https://www.microsoft.com/windows/windows-mixed-reality#specs>

### <a name="what-games-require-a-gamepad-as-opposed-to-motion-controllers"></a>모션 컨트롤러가 아닌 게임 패드가 필요한 게임

**빠른 답변:** 대부분의 몰입형 Windows Mixed Reality 애플리케이션은 모션 컨트롤러를 지원하도록 선택하지만 요구 사항은 아닙니다. 모션 컨트롤러 지원은 환경 개발자에게 달려 있습니다. 개발자는 게임에서 지원하려는 입력 모드를 결정해야 합니다. 스토어에서 특정 게임 또는 앱의 제품 세부 정보 페이지에서 지원되는 컨트롤러 유형을 확인할 수 있습니다.

### <a name="if-i-am-using-xbox-game-streaming-can-i-use-my-xbox-controller"></a>Xbox 게임 스트리밍을 사용하는 경우 Xbox 컨트롤러를 사용할 수 있나요?

**빠른 답변:** 예, Xbox에서 게임을 스트리밍하려면 콘솔에 있을 때와 마찬가지로 Xbox 컨트롤러가 필요합니다.

### <a name="will-windows-mixed-reality-work-if-i-have-no-space-in-my-room"></a>내 공간에 공간이 없으면 Windows Mixed Reality 작동합니까?

**빠른 답변:** 예, 공간이 많지 않은 경우에도 Windows Mixed Reality 작동합니다. "데스크 규모" 환경을 선택하여 설치하는 동안 이를 설정할 수 있습니다.

## <a name="steamvr"></a>SteamVR

### <a name="are-the-minimum-specs-for-steamvr-higher-than-a-windows-mixed-reality-ultra-pc"></a>SteamVR의 최소 사양이 Windows Mixed Reality Ultra PC보다 높나요?

**빠른 답변:** Windows Mixed Reality Ultra PC 요구 사항으로 SteamVR을 실행할 수 있습니다. 그러나 GTX 1070 비디오 카드(또는 그 이상) 및 Intel Core i7 프로세서가 있는 PC에서 SteamVR 미리 보기를 실행하는 것이 좋습니다. 향후 업데이트에서 추가 시스템 구성을 지원하도록 피드백을 계속 검토하고 성능을 최적화하고 있습니다. 이러한 더 높은 사양을 충족하지 않으면 PC에서 Windows Mixed Reality 실행이 차단되지 않지만 전체 환경의 성능과 품질에 영향을 줍니다.

### <a name="are-all-steam-vr-games-be-supported"></a>모든 Steam VR 게임을 지원합니까?

**빠른 답변:** 우리의 의도는 최대한 광범위한 Steam 게임을 지원하는 것이지만, 개발 중에 게임 개발자의 결정에 따라 특정 하드웨어 및 컨트롤러 구성에 따라 Steam 게임을 사용할 수 있다는 점을 명심하세요.
